# MTSD Laboratory work 3 API

A backend part of my simple store project

## Development

1. Create `.mtsd-lab3-api.env` file following [this example](./.mtsd-lab3-api.example.env)
2. Generate Prisma Client:
    ```shell
   npx prisma generate
    ```
3. Start development server
    ```shell
    yarn run start:dev
    ```
